# 🙏 AskPastorApugo AI

An intelligent, Spirit-filled AI assistant designed to provide **biblical answers, prophetic insights, and spiritual guidance** with scripture-based responses.

---

## 🔥 Vision

AskPastorApugo AI is built to:

- Help believers grow spiritually
- Provide instant biblical answers
- Teach the **principles of Jesus**
- Combine **AI + Kingdom wisdom**

---

## ✨ Features

### 🧠 AI Chat System
- Bible-based answers
- Prophetic and Apostolic tones 
- Scripture included in every response

### 🙏 Prayer System
- Automatic prophetic prayers for "how" questions

### 🎤 Sermon Generator
- Full sermon structure:
  - Title
  - Introduction
  - Deep points (with scripture)
  - Application
  - Conclusion

### 🔐 Authentication System
- Login / Register
- Invite code system
- Admin-controlled access

### 👨‍💼 Admin Panel
- User management
- Invite code generation
- System monitoring

### 📊 Dataset System
- Predefined Q&A dataset
- Pending question review
- AI-assisted learning

### ♻️ Trash System
- Deleted data stored safely
- Restore functionality (coming)
